科学早已失声

发表于 2022-10-30 | 分类于 社会 | | 浏览 次
字数统计: 903 | 阅读时长 ≈ 3

我不清楚上面究竟在害怕什么。我最想听到的是科学界的声音，从来没有什么声音会出来讨论这个virus的危害性，我不清楚是根据什么制定出这些policy.

科学界死亡了，我印象中只有一个张文宏医生，在很早之前，3/4月份上海的时候讨论过。当时好像被人骂得很严重。事实是什么，没有人知道，科学的意义是什么，我不清楚。

关于这个virus会对人体健康造成什么样的伤害，为什么讨论这个点的文章基本没有在公众的视野里出现过。事实都没有弄清楚，剩下的对policy的讨论就是没有标准的争吵。

而这些争吵，以我们在媒体上看见的东西为依据。是的，放开将会出现医疗资源挤兑，将会有无数生命逝去，可能就是你家的老人，经济会停摆，在世界格局风云变幻的今天，我们未来要怎么办，我们还有统一的事业，如何如何。这些，有什么依据吗？没有，都是想象。

科学早已失声。

很久之前，我们能听到制造疫苗的好消息，我们速度很快。我们也有一段时间可以看见挺多人热衷于发那些关于打了疫苗的朋友圈。而到了后面，变成了需要用东西奖励那些老人去打疫苗，说是，感染之后能减轻重症风险。是的，科学在这里又一次失声了。

我记得一开始还有一些声音讨论这个疫苗如果用了还会感染，那么它的目的何在？也会看见一些人说自己不愿意去打疫苗，毕竟彼时还有关于疫苗的一些副作用的讨论。而后我看见关于疫苗能对virus起多大作用，给的解释是，能减轻感染后的症状。再后来，没有人在意这个疫苗究竟有没有作用，关于其副作用的声音也不见了。也没有再出现什么关于疫苗讨论的文章了，就是这样慢慢式微，慢慢消失了。

科学失声，媒体一刻不停。

有时候我在想，这些发布消息的人，他们是怎么想的。他们应该知道自己在做什么吧？他们不会为自己有这个能力让一些东西显露出来，却无法做到而感到痛苦吗？我觉得他们会，是个人都应该会，是我可能会辞职，希望他们顶住压力，都是为了混口饭。

我在今年4月上海那个时候对这些事情已经非常的疲惫，很抱歉我没有经邦济世的情怀。但是这些事情如果有一天发生在自己以及家人朋友身上，那么我现在的沉默与悲观就是一种愚蠢。

没有一个可以正大光明讨论的地方，我说过了，government is not God. there is no any need to be perfect. the only one thing they need to do is to be responsible, for things that we truly care. but they still chose to be a perfect god. however, it’s impossible to be one. therefore, they need to hide their mistakes, which are truth.

这不仅会造成很多资源浪费，也会葬送本应拥有的经济收入。对于有些人来说，他们失去了青春，爱人，孩子，甚至是生命，以及本应对未来充满希望的眼光。
